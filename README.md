# SQL Advanced Data Analytics Project — MS SQL + Tableau

## Overview
End-to-end banking analytics using SQL Server and Tableau.
It analyzes 50K transactions and 5K customers to derive
insights into spending patterns, customer tiers, and retention.

## Tech Stack
- SQL Server (T-SQL)
- Tableau Desktop/Public
- Python (for data generation)

## Folder Structure
/data – CSV datasets  
/sql – SQL scripts  
/visuals – Dashboard images  
/Tableau_Project – Tableau workbook

## Tableau Dashboard
View online: [Tableau Public — Customer Transaction Insights](https://public.tableau.com/app/profile/pradeep.kumar1408/vizzes)

## Key Features
- Customer segmentation (Platinum, Gold, Silver, Bronze)
- Monthly transaction trend & MoM growth
- LTV and Cohort analysis
